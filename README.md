A collection of Cocoa Touch classes, extensions and modules to fast track iOS development.

## Installation

### [CocoaPods](http://cocoapods.org)

#### Podfile

```ruby
pod 'Xcore', '~> 1.0'
```

#### Latest version
```ruby
pod 'Xcore', :git => 'https://github.com/zmian/xcore.swift'
```
